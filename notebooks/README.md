# Notebooks

Recommended notebook order:

1. `01_eda.ipynb`: data checks, target distributions, missing values and feature groups.
2. `02_baseline.ipynb`: quick Colab-friendly baseline and first submission.
3. `03_linear_experiments.ipynb`: Ridge, ElasticNet, PLS and target transforms.
4. `04_tree_experiments.ipynb`: ExtraTrees, RandomForest and sklearn boosting.
5. `05_boosting_experiments.ipynb`: CatBoost, LightGBM and XGBoost.
6. `06_ensemble.ipynb`: OOF blending and final submission.
